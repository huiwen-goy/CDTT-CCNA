CDTT-CCNA data analysis (segmented regression)

Note: Data exclusion

  1. Original dataset had n=264.
  2. Dropped single outlier (PTA < 19 & SRT > 2).
  3. Dropped two participants with missing SRT data.
  4. All plots and analyses use n=261 now.
